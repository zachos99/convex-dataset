# CONVEX Dataset & Code

This repository accompanies the paper **“The Synthetic Shift: Tracking the Rise, Virality, and Detectability of AI-Generated Multimodal Misinformation”** and provides code + processed dataset artifacts for analysis and benchmarking.

## What’s in this repo

- **`src/`**: Code for dataset renewal, weakly-supervised misinformation labeling, longitudinal analyses (volume/virality/engagement/consensus/AI references), and benchmark evaluation. Start with `src/README.md`.
- **`data/`**: The two dataset subsets used in the paper (image and video).
- **`supplementary_material/`**: Supplementary material PDF.

## Data note (privacy / availability)

To reduce risk and comply with platform and privacy constraints, this repository **does not include sensitive or X-extracted content**. Some tweet-level fields may appear in anonymized or non-content form (engagement counts). Any additional tweet-level content or media that is not part of the public Community Notes downloads should be **reconstructed by users independently** (e.g., by downloading Community Notes checkpoints and re-running the pipeline) or **requested by contacting the authors**, subject to applicable policies and approvals.
